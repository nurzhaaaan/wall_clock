# Analog Clock Python

A simple and elegant analog clock application built with **Python** and **Tkinter**. It displays the current time with hour, minute, and second hands, featuring a clean user interface.

## Features
* **Real-time Updates:** Synchronizes with your system time to update the clock hands every second.
* **Graphical Interface:** Uses Python's built-in `tkinter` library to render the clock face.
* **Responsive Design:** Precise mathematical calculations (sin/cos) ensure the hands point accurately to their respective positions.
